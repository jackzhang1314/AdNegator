## 1. API 概述

```
Production:  https://api.gvisor-e2b.com
Staging:     https://api-staging.gvisor-e2b.com
Development: http://localhost:8000
```

## 2. 认证方式 (E2B SDK v2.6.2 Compatible)

### 2.1 双重认证机制

SDK支持两种认证方式，按以下优先级检查:
1. **API Key** (团队/组织级别) - 用于创建和管理sandbox
2. **Access Token** (个人/用户级别) - 用于个人sandbox操作

### 2.2 API Key认证

**适用场景**: Sandbox创建、模板管理、配额查询

**HTTP Header**:
```
X-API-Key: sk_abc123...
```

**环境变量**:
```bash
export E2B_API_KEY=sk_abc123...
```

**获取方式**:
1. 注册账户
2. 在控制台生成 API Key
3. 设置作用域 (scopes)

**示例**:
```bash
curl -X POST https://api.gvisor-e2b.com/sandboxes \
  -H "X-API-Key: sk_abc123..." \
  -H "Content-Type: application/json" \
  -d '{"templateID": "python-3.11"}'
```

**TypeScript SDK**:
```typescript
import { Sandbox } from '@adnegator/sdk'

const sandbox = await Sandbox.create('base', {
  apiKey: 'sk_abc123...'  // 或设置环境变量 E2B_API_KEY
})
```

### 2.3 Access Token认证

**适用场景**: 个人sandbox操作

**HTTP Header**:
```
Authorization: Bearer eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...
```

**环境变量**:
```bash
export E2B_ACCESS_TOKEN=eyJhbGciOi...
```

**示例**:
```bash
curl -X GET https://api.gvisor-e2b.com/sandboxes \
  -H "Authorization: Bearer eyJhbGciOi..." \
  -H "Content-Type: application/json"
```

**TypeScript SDK**:
```typescript
import { Sandbox } from '@adnegator/sdk'

const sandbox = await Sandbox.create('base', {
  accessToken: 'eyJhbGciOi...'  // 或设置环境变量 E2B_ACCESS_TOKEN
})
```

### 2.4 Data Plane认证 (envdAccessToken)

**适用场景**: gRPC操作 (命令执行、文件系统)

**gRPC Metadata Header**:
```
Authorization: Basic base64(username:)
```

**Token 来源**: 创建沙盒时返回的 `envdAccessToken`

**认证格式**:
- 使用HTTP Basic Auth格式
- username为用户名 (默认: 'user')
- password为空
- Base64编码后放入Authorization header

**示例**:
```javascript
import { createConnectTransport } from '@connectrpc/connect-web'

// 生成认证header
function authenticationHeader(username = 'user') {
  const credentials = `${username}:`
  const base64Credentials = btoa(credentials)
  return `Basic ${base64Credentials}`
}

const client = createConnectTransport({
  baseUrl: `https://${sandboxDomain}:49983`,
  httpVersion: "2",
  headers: {
    Authorization: authenticationHeader('user')
  }
})
```

**Python SDK**:
```python
import base64

def authentication_header(username: str = 'user') -> str:
    """生成gRPC认证header"""
    credentials = f"{username}:"
    base64_credentials = base64.b64encode(credentials.encode()).decode()
    return f"Basic {base64_credentials}"

# 使用
headers = {
    'Authorization': authentication_header('user')
}
```

### 2.5 认证检查顺序

SDK按以下顺序查找凭证:
1. 检查方法参数 (`apiKey` or `accessToken`)
2. 检查环境变量 `E2B_API_KEY`
3. 检查环境变量 `E2B_ACCESS_TOKEN`
4. 如果都不存在，抛出 `AuthenticationError`

**业务规则**: BR-012 (API Key验证)

### 2.6 认证错误响应

**HTTP 401 - Unauthorized**:
```json
{
  "error": {
    "type": "AuthenticationError",
    "code": "unauthorized",
    "message": "Invalid or expired API key"
  }
}
```

**gRPC UNAUTHENTICATED**:
```json
{
  "code": "unauthenticated",
  "message": "Invalid credentials",
  "details": [{
    "type": "google.rpc.ErrorInfo",
    "reason": "INVALID_CREDENTIALS",
    "metadata": {
      "error_type": "AuthenticationError"
    }
  }]
}
```

## 4. 沙盒管理 API

### 4.1 创建沙盒

**Endpoint**: `POST /sandboxes`

**描述**: 创建新的代码执行沙盒

**对应需求**: L1-F1.1 创建沙盒
**对应业务规则**: BR-020, BR-021, BR-022

**请求体**:
```json
{
  "templateID": "python-3.11",      // 可选，默认 "base"
  "timeout": 3600,                   // 可选，默认 3600 秒
  "metadata": {                      // 可选
    "project": "AI Agent",
    "env": "dev"
  },
  "secure": true,                    // E2B 兼容，生成 envdAccessToken
  "autoPause": false                 // E2B 兼容字段（接受但忽略，E2B 不支持 pause/resume）
}
```

**响应** (201 Created):
```json
{
  "sandboxID": "sbx_a1b2c3d4e5f6g7h8",
  "envdVersion": "0.2.0",
  "envdAccessToken": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...",
  "domain": "sbx-a1b2c3d4e5f6g7h8.sandboxes.gvisor-e2b.com"
}
```

**cURL 示例**:
```bash
curl -X POST https://api.gvisor-e2b.com/sandboxes \
  -H "X-API-Key: sk_abc123..." \
  -H "Content-Type: application/json" \
  -d '{
    "templateID": "python-3.11",
    "timeout": 3600,
    "metadata": {"project": "test"}
  }'
```

**错误码**:
- `400` - 参数错误 (invalid_parameters)
- `401` - 认证失败 (unauthorized)
- `403` - 配额超限 (quota_exceeded)
- `500` - 服务器错误 (internal_error)

---

### 4.2 获取沙盒信息

**Endpoint**: `GET /sandboxes/{sandboxID}`

**描述**: 查询沙盒详细信息

**路径参数**:
- `sandboxID` (string, required): 沙盒 ID

**响应** (200 OK):
```json
{
  "sandboxID": "sbx_a1b2c3d4e5f6g7h8",
  "status": "running",
  "templateID": "python-3.11",
  "domain": "sbx-a1b2c3d4e5f6g7h8.sandboxes.gvisor-e2b.com",
  "createdAt": "2025-11-05T12:00:00.000Z",
  "updatedAt": "2025-11-05T12:00:05.000Z",
  "metadata": {
    "project": "test"
  }
}
```

**状态说明** (E2B 官方 4 态模型):
- `creating`: 创建中（Firecracker microVM 启动中）
- `running`: 运行中（envd 可访问）
- `terminating`: 销毁中（正在清理资源）
- (deleted): 已销毁（数据库记录已删除，404 响应）

**注意**: E2B **不支持** pause/resume 功能，沙盒生命周期仅包含上述 3 个可查询状态

**错误码**:
- `404` - 沙盒不存在 (sandbox_not_found)

---

### 4.3 列出沙盒

**Endpoint**: `GET /sandboxes`

**描述**: 查询用户的沙盒列表

**查询参数**:
- `status` (string, optional): 状态筛选 (creating, running, terminating)
- `page` (integer, optional): 页码，默认 1
- `page_size` (integer, optional): 每页数量，默认 20

**响应** (200 OK):
```json
{
  "data": [
    {
      "sandboxID": "sbx_xxx",
      "status": "running",
      "templateID": "python-3.11",
      "createdAt": "2025-11-05T12:00:00.000Z"
    }
  ],
  "pagination": {
    "page": 1,
    "page_size": 20,
    "total": 5,
    "total_pages": 1
  }
}
```

---

### 4.4 删除沙盒

**Endpoint**: `DELETE /sandboxes/{sandboxID}`

**描述**: 删除沙盒并清理所有资源

**对应需求**: L1-F1.4 销毁沙盒

**路径参数**:
- `sandboxID` (string, required): 沙盒 ID

**响应** (204 No Content):
```
(空响应体)
```

**清理内容** (同步执行, <425ms):
- 停止所有运行中的进程
- 停止 Firecracker microVM
- 删除 Nomad Job
- 删除数据库记录（硬删除）

**对应业务规则**: BR-110 (同步删除), BR-033 (清理流程)

**错误码**:
- `404` - 沙盒不存在 (sandbox_not_found)

**性能保证**:
- 删除操作同步完成，总耗时 <425ms
- 幂等操作：重复删除返回 204

---

### 4.5 暂停沙盒 (Beta)

**Endpoint**: `POST /sandboxes/{sandboxID}/pause`

**描述**: 暂停沙盒并保存状态（文件系统+内存+进程）

**功能状态**: ✅ Beta (生产可用，免费)

**路径参数**:
- `sandboxID` (string, required): 沙盒 ID

**响应** (204 No Content):
```
(空响应体)
```

**保存内容**:
- ✅ 完整文件系统
- ✅ 内存状态
- ✅ 所有运行中的进程
- ✅ 已加载的变量和数据

**性能**:
- 暂停时间: ~4秒/GB RAM
- 数据保留: 最多30天

**错误码**:
- `404` - 沙盒不存在 (sandbox_not_found)
- `409` - 沙盒已暂停 (already_paused)

**注意**: 暂停后沙盒的网络端口将不可访问

---

### 4.6 连接/恢复沙盒

**Endpoint**: `POST /sandboxes/{sandboxID}/connect`

**描述**: 连接到现有沙盒，如果已暂停则自动恢复

**路径参数**:
- `sandboxID` (string, required): 沙盒 ID

**请求体**:
```json
{
  "timeout": 3600  // 超时时间（秒），从当前时间开始计算
}
```

**响应** (200 OK - 如果running / 201 Created - 如果paused后恢复):
```json
{
  "sandboxID": "sbx_a1b2c3d4e5f6g7h8",
  "status": "running",
  "envdAccessToken": "eyJhbGciOiJSUzI1NiIsInR5cCI6IkpXVCJ9...",
  "domain": "sbx-a1b2c3d4e5f6g7h8.sandboxes.gvisor-e2b.com"
}
```

**性能**:
- 恢复时间: ~1秒（从 paused 恢复）
- 连接时间: <50ms（如果已 running）

**错误码**:
- `404` - 沙盒不存在或已过期 (sandbox_not_found)

**注意**:
- 这是推荐的恢复方式（`POST /sandboxes/{sandboxID}/resume` 已废弃）
- 可用于延长 running 状态沙盒的 TTL

## 6. envd Data Plane API (Connect RPC v2.0.0)

### 6.1 协议说明

envd使用 **Connect RPC** (gRPC over HTTP/2) 进行实时双向通信

**核心特性**:
- Protocol Buffers序列化
- HTTP/2双向流
- 类型安全的接口
- 跨语言支持 (TypeScript, Python, Go)

**依赖版本**:
- Connect RPC: v2.0.0-rc.3+ (TypeScript)
- Protobuf: v2.6.2+ (@bufbuild/protobuf)
- e2b_connect: custom (Python)

**Base URL**:
```
https://${sandboxDomain}:49983
```

### 6.2 Process Service (process.proto)

**完整Protobuf定义**:
```protobuf
syntax = "proto3";

package process;

// Process服务 - 命令执行和进程管理
service Process {
  // 列出所有进程
  rpc List(ListRequest) returns (ListResponse);

  // 启动新进程 (返回流式输出)
  rpc Start(StartRequest) returns (stream StartResponse);

  // 连接到运行中的进程
  rpc Connect(ConnectRequest) returns (stream ConnectResponse);

  // 发送输入数据到进程stdin
  rpc SendInput(SendInputRequest) returns (SendInputResponse);

  // 发送信号到进程
  rpc SendSignal(SendSignalRequest) returns (SendSignalResponse);

  // 更新PTY大小 (终端尺寸)
  rpc Update(UpdateRequest) returns (UpdateResponse);
}

// ========== 列出进程 ==========

message ListRequest {
  // 空请求
}

message ListResponse {
  repeated ProcessConfig processes = 1;
}

message ProcessConfig {
  int64 pid = 1;                      // 进程ID
  string tag = 2;                     // 自定义标签 (可选)
  string cmd = 3;                     // 命令
  repeated string args = 4;           // 参数列表
  map<string, string> envs = 5;       // 环境变量
  string cwd = 6;                     // 工作目录
}

// ========== 启动进程 ==========

message StartRequest {
  string cmd = 1;                     // 命令 (必填)
  repeated string args = 2;           // 参数列表
  map<string, string> envs = 3;       // 环境变量
  string cwd = 4;                     // 工作目录
  string tag = 5;                     // 自定义标签
  PtySize pty_size = 6;               // PTY尺寸 (终端模式)
  string user = 7;                    // 执行用户 (默认: 'user')
  int64 timeout_ms = 8;               // 超时时间 (毫秒)
  bool background = 9;                // 后台运行
}

message PtySize {
  int32 cols = 1;                     // 列数 (默认: 80)
  int32 rows = 2;                     // 行数 (默认: 24)
}

message StartResponse {
  oneof event {
    ProcessStarted started = 1;       // 进程已启动
    ProcessOutput output = 2;         // 进程输出
    ProcessExited exited = 3;         // 进程已退出
  }
}

message ProcessStarted {
  int64 pid = 1;                      // 进程ID
}

message ProcessOutput {
  enum OutputType {
    STDOUT = 0;                       // 标准输出
    STDERR = 1;                       // 标准错误
    PTY = 2;                          // PTY输出 (混合)
  }

  OutputType type = 1;
  bytes data = 2;                     // 输出数据
  int64 timestamp = 3;                // 时间戳 (毫秒)
}

message ProcessExited {
  int32 exit_code = 1;                // 退出码
  string error = 2;                   // 错误信息 (如果失败)
}

// ========== 连接到进程 ==========

message ConnectRequest {
  int64 pid = 1;                      // 进程ID
}

message ConnectResponse {
  oneof event {
    ProcessOutput output = 1;         // 进程输出
    ProcessExited exited = 2;         // 进程已退出
  }
}

// ========== 发送输入 ==========

message SendInputRequest {
  int64 pid = 1;                      // 进程ID
  bytes data = 2;                     // 输入数据
}

message SendInputResponse {
  // 空响应
}

// ========== 发送信号 ==========

message SendSignalRequest {
  int64 pid = 1;                      // 进程ID
  Signal signal = 2;                  // 信号类型
}

enum Signal {
  SIGTERM = 0;                        // 终止 (15)
  SIGKILL = 1;                        // 强制终止 (9)
}

message SendSignalResponse {
  bool success = 1;                   // 是否成功
}

// ========== 更新PTY尺寸 ==========

message UpdateRequest {
  int64 pid = 1;                      // 进程ID
  PtySize pty_size = 2;               // 新的PTY尺寸
}

message UpdateResponse {
  // 空响应
}
```

### 6.3 Filesystem Service (filesystem.proto)

**完整Protobuf定义**:
```protobuf
syntax = "proto3";

package filesystem;

// Filesystem服务 - 文件系统操作
service Filesystem {
  // 获取文件/目录信息
  rpc Stat(StatRequest) returns (StatResponse);

  // 列出目录内容
  rpc ListDir(ListDirRequest) returns (ListDirResponse);

  // 创建目录
  rpc MakeDir(MakeDirRequest) returns (MakeDirResponse);

  // 移动/重命名文件
  rpc Move(MoveRequest) returns (MoveResponse);

  // 删除文件/目录
  rpc Remove(RemoveRequest) returns (RemoveResponse);

  // 监视目录变化 (流式)
  rpc WatchDir(WatchDirRequest) returns (stream WatchDirResponse);

  // 创建监视器 (非流式, 用于轮询)
  rpc CreateWatcher(CreateWatcherRequest) returns (CreateWatcherResponse);

  // 获取监视器事件
  rpc GetWatcherEvents(GetWatcherEventsRequest) returns (GetWatcherEventsResponse);

  // 删除监视器
  rpc RemoveWatcher(RemoveWatcherRequest) returns (RemoveWatcherResponse);
}

// ========== 文件信息 ==========

message EntryInfo {
  string path = 1;                    // 完整路径
  string name = 2;                    // 文件名
  FileType type = 3;                  // 文件类型
  int64 size = 4;                     // 大小 (字节)
  uint32 mode = 5;                    // 权限位 (Unix)
  string permissions = 6;             // 权限字符串 (如 'rwxr-xr-x')
  string owner = 7;                   // 所有者
  string group = 8;                   // 所有组
  int64 modified_time = 9;            // 修改时间 (Unix时间戳)
  string symlink_target = 10;         // 符号链接目标 (如果是链接)
}

enum FileType {
  FILE = 0;                           // 普通文件
  DIR = 1;                            // 目录
}

// ========== Stat (获取文件信息) ==========

message StatRequest {
  string path = 1;                    // 文件路径
  string user = 2;                    // 执行用户
}

message StatResponse {
  EntryInfo entry = 1;                // 文件信息
}

// ========== ListDir (列出目录) ==========

message ListDirRequest {
  string path = 1;                    // 目录路径
  string user = 2;                    // 执行用户
}

message ListDirResponse {
  repeated EntryInfo entries = 1;     // 目录内容列表
}

// ========== MakeDir (创建目录) ==========

message MakeDirRequest {
  string path = 1;                    // 目录路径
  string user = 2;                    // 执行用户
  uint32 mode = 3;                    // 权限 (默认: 0755)
}

message MakeDirResponse {
  // 空响应
}

// ========== Move (移动/重命名) ==========
```

### 6.4 版本兼容性检查

**版本检查函数**:
```typescript
function compareVersions(v1: string, v2: string): number {
  const parts1 = v1.split('.').map(Number)
  const parts2 = v2.split('.').map(Number)

  for (let i = 0; i < 3; i++) {
    if (parts1[i] > parts2[i]) return 1
    if (parts1[i] < parts2[i]) return -1
  }
  return 0
}

// 使用示例
const sandboxInfo = await sandbox.getInfo()
const supportsRecursiveWatch = compareVersions(
  sandboxInfo.envdVersion,
  ENVD_VERSION_RECURSIVE_WATCH
) >= 0

if (!supportsRecursiveWatch) {
  throw new TemplateError(
    'Recursive watch requires envd >= 0.1.4. ' +
    `Current version: ${sandboxInfo.envdVersion}`
  )
}
```

### A. SDK 使用示例

#### TypeScript SDK

```typescript
import { Sandbox } from '@gvisor-e2b/sdk';

// 配置 Base URL
const apiUrl = 'https://api.gvisor-e2b.com';
const apiKey = process.env.API_KEY;

// 创建沙盒
const sandbox = await Sandbox.create({
  apiUrl,
  apiKey,
  template: 'python-3.11',
  timeoutMs: 3600000  // 1 hour
});

// 执行代码
const proc = await sandbox.commands.start({
  cmd: 'python',
  args: ['script.py']
});

// 实时输出
proc.stdout.on('data', (data) => {
  console.log(data.toString());
});

await proc.wait();

// 清理
await sandbox.kill();
```

```python
from gvisor_e2b import Sandbox

# 创建沙盒
sandbox = Sandbox.create(
```
