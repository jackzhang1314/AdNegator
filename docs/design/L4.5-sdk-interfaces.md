# L4.5 SDK接口规范 (SDK Interface Specification)

## 文档概述
本文档定义AdNegator平台与E2B官方SDK 100%兼容的接口规范，包括TypeScript/JavaScript和Python两种语言的SDK实现标准。

**版本兼容目标**: E2B SDK v2.6.2

---

## 1. SDK架构概述

### 1.1 TypeScript/JavaScript SDK

**运行时支持**:
- Node.js (>= 20)
- Deno (with --allow flags)
- Bun
- Browser (limited: no template building, no glob)
- Vercel Edge (partial)
- Cloudflare Workers (partial)

**核心模块**:
```
src/
├── api/              # REST/OpenAPI客户端
├── envd/             # gRPC客户端 (protobuf)
├── sandbox/          # Sandbox核心类
├── template/         # 模板构建
├── connectionConfig.ts
└── index.ts          # 主导出
```

### 1.2 Python SDK

**Python版本**: 3.9+

**架构**:
```
e2b/
├── api/              # REST客户端
├── envd/             # gRPC客户端
├── sandbox_sync/     # 同步API
├── sandbox_async/    # 异步API
├── template_sync/    # 同步模板API
├── template_async/   # 异步模板API
└── connection_config.py
```

### 2.2 Python接口

```python
class ApiParams(TypedDict):
    """API连接参数"""
    api_key: Optional[str]
    access_token: Optional[str]
    domain: Optional[str]
    api_url: Optional[str]
    debug: Optional[bool]
    request_timeout: Optional[float]  # 秒

```

### 2.3 认证优先级

检查顺序:
1. **API Key** (`X-API-Key` header)
   - 用于创建和管理sandbox
   - 团队/组织级别权限
2. **Access Token** (`Authorization: Bearer` header)
   - 用于个人sandbox操作
   - 用户级别权限

**实现规则**:
- 优先检查API Key
- 如果API Key不存在，检查Access Token
- 至少需要一种认证方式

### 6.3 签名算法

**签名生成** (HMAC-SHA256):
```typescript
/**
 * 生成URL签名
 * @param secret 密钥
 * @param sandboxId Sandbox ID
 * @param path 文件路径
 * @param operation 操作 ('read' | 'write')
 * @param user 用户名
 * @param expiration 过期时间戳 (可选)
 */
function getSignature(
  secret: string,
  sandboxId: string,
  path: string,
  operation: 'read' | 'write',
  user: string,
  expiration?: number
): string {
  const message = [
    sandboxId,
    path,
    operation,
    user,
    expiration?.toString() || ''
  ].join('.')

  const signature = hmac_sha256(secret, message)
  const signatureB64 = base64_urlsafe_encode(signature).replace(/=+$/, '')

  return `v1,${signatureB64}`
}
```

class NotFoundException(SandboxException):
    """资源未找到异常"""
    pass

class AuthenticationException(SandboxException):
    """认证失败异常"""
    pass

class TemplateException(SandboxException):
    """模板错误异常"""
    pass

class RateLimitException(SandboxException):
    """速率限制异常"""
    retry_after: Optional[int]  # 秒

class BuildException(SandboxException):
    """构建失败异常"""
    pass

class FileUploadException(SandboxException):
    """文件上传失败异常"""
    pass

    """命令退出异常"""

    stdout: str
    stderr: str
    error: Optional[str]
