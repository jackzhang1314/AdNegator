## 1. 业务规则概述

### 1.1 业务规则分类

| 分类 | 描述 | 优先级 |
|------|------|--------|
| **强制规则 (Hard Rules)** | 必须满足，违反则拒绝操作 | P0 |
| **软规则 (Soft Rules)** | 建议遵守，可配置例外 | P1 |
| **策略规则 (Policy Rules)** | 可动态调整的业务策略 | P2 |

### 1.2 规则编号约定

**格式**: `BR-{类别}{序号}`

| 类别代码 | 类别名称 | 示例 |
|----------|----------|------|
| **00x** | 用户管理 | BR-001 |
| **01x** | 认证授权 | BR-010 |
| **02x** | 沙盒生命周期 | BR-020 |
| **03x** | 资源配额 | BR-030 |
| **04x** | 进程执行 | BR-040 |
| **05x** | 文件系统 | BR-050 |
| **06x** | 暂停恢复 | BR-060 |
| **07x** | 超时清理 | BR-070 |
| **08x** | 计费 | BR-080 |
| **09x** | 安全 | BR-090 |
| **10x** | 数据生命周期 | BR-100 |

## 2. 用户管理规则

### BR-001: 邮箱唯一性

**规则类型**: 强制规则
**描述**: 每个邮箱只能注册一个账户

**实现**:
```python
async def register_user(email: str, name: str, password: str):
    # 检查邮箱是否已存在
    existing_user = await db.query(User).filter(
        User.email == email,
        User.deleted_at.is_(None)
    ).first()

    if existing_user:
        raise BusinessRuleViolation(
            code="BR-001",
            message="Email already registered"
        )

    # 创建用户
    user = User(email=email, name=name, password_hash=hash_password(password))
    await db.add(user)
    await db.commit()
```

**数据库约束** (L3.2):
```sql
ALTER TABLE users ADD CONSTRAINT users_email_unique UNIQUE (email);
```

---

### BR-002: 密码强度要求

**规则类型**: 强制规则
**描述**: 密码必须满足以下条件：
- 最小长度 8 字符
- 包含至少 1 个大写字母
- 包含至少 1 个小写字母
- 包含至少 1 个数字

**实现**:
```python
import re

def validate_password_strength(password: str) -> bool:
    if len(password) < 8:
        raise BusinessRuleViolation(
            code="BR-002",
            message="Password must be at least 8 characters"
        )

    if not re.search(r'[A-Z]', password):
        raise BusinessRuleViolation(
            code="BR-002",
            message="Password must contain at least one uppercase letter"
        )

    if not re.search(r'[a-z]', password):
        raise BusinessRuleViolation(
            code="BR-002",
            message="Password must contain at least one lowercase letter"
        )

    if not re.search(r'\d', password):
        raise BusinessRuleViolation(
            code="BR-002",
            message="Password must contain at least one digit"
        )

    return True
```

## 3. 认证授权规则

### BR-010: API Key 格式

**规则类型**: 强制规则
**描述**: API Key 必须符合格式 `sk_[a-z0-9]{40}`

**实现**:
```python
import secrets

def generate_api_key() -> str:
    """生成符合规范的 API Key"""
    random_part = secrets.token_hex(20)  # 40 字符
    return f"sk_{random_part}"

def validate_api_key_format(key: str) -> bool:
    pattern = r'^sk_[a-z0-9]{40}$'
    if not re.match(pattern, key):
        raise BusinessRuleViolation(
            code="BR-010",
            message="Invalid API key format"
        )
    return True
```

---

### BR-011: API Key 数量限制

**规则类型**: 软规则
**描述**: 每个用户最多创建 10 个 API Key

**配置**:
```python
MAX_API_KEYS_PER_USER = 10  # 可通过环境变量配置
```

**实现**:
```python
async def create_api_key(user_id: UUID, name: str, scopes: List[str]):
    # 统计当前 API Key 数量
    count = await db.query(ApiKey).filter(
        ApiKey.user_id == user_id,
        or_(ApiKey.expires_at.is_(None), ApiKey.expires_at > datetime.utcnow())
    ).count()

    if count >= MAX_API_KEYS_PER_USER:
        raise BusinessRuleViolation(
            code="BR-011",
            message=f"Maximum {MAX_API_KEYS_PER_USER} API keys per user"
        )

    # 创建 API Key
    key = generate_api_key()
    api_key = ApiKey(user_id=user_id, key=key, name=name, scopes=scopes)
    await db.add(api_key)
    await db.commit()
```

### BR-012: API Key 过期检查

**规则类型**: 强制规则
**描述**: 使用 API Key 时必须检查是否过期

**实现**:
```python
async def verify_api_key(key: str) -> User:
    api_key = await db.query(ApiKey).filter(ApiKey.key == key).first()

    if not api_key:
        raise AuthenticationError("Invalid API key")

    # 检查过期时间
    if api_key.expires_at and api_key.expires_at < datetime.utcnow():
        raise BusinessRuleViolation(
            code="BR-012",
            message="API key has expired"
        )

    # 更新最后使用时间
    api_key.last_used_at = datetime.utcnow()
    await db.commit()

    return api_key.user
```

### BR-013: 权限范围验证

**规则类型**: 强制规则
**描述**: API 操作必须在 API Key 的 scopes 范围内

**Scope 定义**:
```python
VALID_SCOPES = [
    "sandbox:create",
    "sandbox:read",
    "sandbox:pause",
    "sandbox:delete",
    "sandbox:*",        # 所有沙盒权限
    "template:create",
    "template:read",
    "template:*",
    "*"                 # 所有权限
]
```

**实现**:
```python
def check_scope(api_key: ApiKey, required_scope: str) -> bool:
    """检查 API Key 是否有所需权限"""
    scopes = api_key.scopes or []

    # 检查通配符
    if "*" in scopes:
        return True

    # 检查资源级通配符（如 sandbox:*）
    resource = required_scope.split(':')[0]
    if f"{resource}:*" in scopes:
        return True

    # 检查精确匹配
    if required_scope in scopes:
        return True

    raise BusinessRuleViolation(
        code="BR-013",
        message=f"Insufficient permissions: {required_scope} required"
    )
```

### BR-023: 状态转换规则

**规则类型**: 强制规则
**描述**: 沙盒状态转换必须遵循有效路径（详见 L4.2 状态图）

**有效转换**:
```python
VALID_STATE_TRANSITIONS = {
    'creating': ['running', 'failed'],
    'running': ['pausing', 'terminating'],
    'pausing': ['paused', 'failed'],
    'paused': ['resuming', 'terminating'],
    'resuming': ['running', 'failed'],
    'terminating': [],  # 终态
    'failed': ['terminating']  # 可清理
}
```

**实现**:
```python
async def update_sandbox_status(sandbox_id: str, new_status: str):
    sandbox = await db.query(Sandbox).filter(
        Sandbox.sandbox_id == sandbox_id
    ).first()

    current_status = sandbox.status

    # 检查转换是否合法
    valid_transitions = VALID_STATE_TRANSITIONS.get(current_status, [])
    if new_status not in valid_transitions:
        raise BusinessRuleViolation(
            code="BR-023",
            message=f"Invalid state transition: {current_status} -> {new_status}"
        )

    # 更新状态
    sandbox.status = new_status
    sandbox.updated_at = datetime.utcnow()
    await db.commit()
```

### BR-031: 内存配额范围

**规则类型**: 强制规则
**描述**: 沙盒内存配额范围 512Mi-32Gi

**配置**:
```python
MEMORY_LIMITS = {
    "min": "512Mi",
    "max": "32Gi",
    "default": "4Gi"
}
```

**实现**:
```python
def parse_memory(memory_str: str) -> int:
    """解析内存字符串为字节数"""
    units = {'Ki': 1024, 'Mi': 1024**2, 'Gi': 1024**3}
    match = re.match(r'(\d+)(Ki|Mi|Gi)', memory_str)
    if not match:
        raise ValueError("Invalid memory format")
    value, unit = match.groups()
    return int(value) * units[unit]

def validate_memory_quota(memory: str):
    memory_bytes = parse_memory(memory)
    min_bytes = parse_memory(MEMORY_LIMITS["min"])
    max_bytes = parse_memory(MEMORY_LIMITS["max"])

    if memory_bytes < min_bytes or memory_bytes > max_bytes:
        raise BusinessRuleViolation(
            code="BR-031",
            message=f"Memory must be between {MEMORY_LIMITS['min']} and {MEMORY_LIMITS['max']}"
        )
```

    defer cancel()

func validatePath(path string) error {
    forbiddenPaths := []string{"/etc", "/sys", "/proc", "/root", "/var"}

    for _, forbidden := range forbiddenPaths {
        if strings.HasPrefix(path, forbidden) {
            return status.Errorf(
                codes.PermissionDenied,
                "BR-051: Access to %s is forbidden",
                path,
            )

        return status.Errorf(
            codes.PermissionDenied,
            "BR-051: Path must be under /workspace",
        )
    }

    return nil
}
```

async def create_checkpoint(sandbox_id: UUID, checkpoint_url: str):
    # 统计现有检查点
    checkpoint_count = await db.query(Checkpoint).filter(
        Checkpoint.sandbox_id == sandbox_id
    ).count()

    if checkpoint_count >= 3:
        # 删除最旧的检查点
        oldest = await db.query(Checkpoint).filter(
            Checkpoint.sandbox_id == sandbox_id
        ).order_by(Checkpoint.created_at.asc()).first()

        await s3_client.delete_object(oldest.checkpoint_url)
        await db.delete(oldest)

    # 创建新检查点
    checkpoint = Checkpoint(
        sandbox_id=sandbox_id,
        checkpoint_url=checkpoint_url,
        size_bytes=size,
        checksum=checksum
    )
    await db.add(checkpoint)
    await db.commit()
```

## 10. 计费规则

### BR-080: 计费单位

**规则类型**: 策略规则
**描述**: 沙盒按分钟计费，不足 1 分钟按 1 分钟计

**计费公式**:
```python
def calculate_cost(sandbox: Sandbox) -> Decimal:
    """
    计费公式:
    cost = ceil(runtime_seconds / 60) * rate_per_minute * cpu_multiplier * memory_multiplier
    """
    runtime_seconds = (sandbox.deleted_at or datetime.utcnow()) - sandbox.created_at
    runtime_minutes = math.ceil(runtime_seconds.total_seconds() / 60)

    # 基础费率（每分钟）
    base_rate = Decimal('0.01')  # $0.01/min

    # CPU 系数
    cpu = sandbox.resources.get('cpu', 2)
    cpu_multiplier = Decimal(str(cpu / 2))  # 基准 2 核

    # 内存系数
    memory_gb = parse_memory(sandbox.resources.get('memory', '4Gi')) / (1024**3)
    memory_multiplier = Decimal(str(memory_gb / 4))  # 基准 4GB

    cost = runtime_minutes * base_rate * cpu_multiplier * memory_multiplier

    return cost.quantize(Decimal('0.001'))  # 精确到 0.001 美元
```

### BR-081: 免费额度

**规则类型**: 策略规则
**描述**: 免费用户每月 100 分钟免费额度

**配置**:
```python
FREE_TIER_MINUTES = {
    "free": 100,      # 100 分钟/月
    "pro": 10000,     # 10000 分钟/月
    "enterprise": -1  # 无限制
}
```

**实现**:
```python
async def check_quota_before_create(user_id: UUID):
    user = await db.get(User, user_id)
    plan = user.metadata.get('plan', 'free')
    free_minutes = FREE_TIER_MINUTES[plan]

    if free_minutes == -1:
        return  # 无限制

    # 统计本月使用分钟数
    month_start = datetime.utcnow().replace(day=1, hour=0, minute=0, second=0)

    total_minutes = await db.query(
        func.sum(
            func.ceil(
                func.extract('epoch', Sandbox.deleted_at - Sandbox.created_at) / 60
            )
        )
    ).filter(
        Sandbox.user_id == user_id,
        Sandbox.created_at >= month_start
    ).scalar() or 0

    if total_minutes >= free_minutes:
        raise BusinessRuleViolation(
            code="BR-081",
            message=f"Monthly quota exceeded: {total_minutes}/{free_minutes} minutes used"
        )
```

    old_partitions = await db.execute("""
